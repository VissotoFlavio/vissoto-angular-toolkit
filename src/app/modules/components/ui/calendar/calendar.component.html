Exemplo para personalização do Tema

Tema

O parâmetro "Theme" permite selecionar os temas "Classico" e "Dark".

Por padrão, o tema classico será selecionado.

Adicionar Notificação

O parâmetro "notifications" recebe uma coleção do tipo "NotificationCalendar" para que seja adicionado no calendário os dias com notificação

O parâmetro "addNotification" recebe um EventEmitter<NotificationCalendar[]> para que seja adicionado no calendário os dias com notificação

Personalização dos Meses

O parâmetro "months" recebe uma coleção do tipo "Month" Que sera utilizado para criação dos meses.

Por padrão, os meses estão em Português Brasil.
