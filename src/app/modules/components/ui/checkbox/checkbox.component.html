UI Checkbox

Exemplos para utilização do componente Checkbox.

Use o componente select para permitir o usuário selecionar uma opção.

Configuração

Após instalar o componente, será necessário importar o Modulo do Select nos modulos da aplicação, conforme exemplo abaixo.

Eventos disponíveis no componente Select

Exemplo para utilização da propriedade text

O parâmetro "text" recebe um string.

Exemplo para utilização da propriedade checked

O parâmetro "checked" recebe um boolean, para selecionar o elemento.

Por padrão o valor será false.

Exemplo para utilização da propriedade disabled

O parâmetro "disabled" recebe um boolean, que desativa o elemento.

Por padrão o valor será false.

Exemplo para utilização da propriedade HelperText

O parâmetro "helperText" recebe um string, que será exibido logo abaixo do texto.

Exemplo para utilização da propriedade FormControl

Caso esteja utilizando o modulo "ReactiveFormsModule", é possível passar um FormControl para o componente Checkbox.

O parâmetro "control" recebe um objeto "FormControl.
